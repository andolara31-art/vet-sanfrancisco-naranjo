# Clínica Veterinaria San Francisco de Asís

Sitio web oficial de la **Clínica Veterinaria San Francisco de Asís** en Naranjo, Alajuela.
"El bienestar animal del cantón de Naranjo."

## Stack

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS** con paleta veterinaria (verde `#2E7D32`, azul confianza `#1565C0`, beige `#FFF8E7`)
- **Framer Motion** para animaciones (parallax, scroll reveal, micro-interactions)
- **Lucide React** para iconografía
- **Web3Forms** para el formulario de contacto
- Tipografía **Playfair Display** (titulares) + **DM Sans** (cuerpo)

## Secciones

1. Navbar fijo con teléfono visible y CTA de agenda
2. Hero con parallax y tarjetas flotantes
3. 6 servicios clínicos con iconos y hover animado
4. ¿Por qué elegirnos? con estadísticas reales
5. Horarios + ubicación con marcador animado
6. Galería con grid asimétrico
7. Testimonios con ratings
8. CTA de WhatsApp con fondo gradiente
9. Formulario Web3Forms (nombre, teléfono, mascota, motivo, mensaje)
10. Footer + botón WhatsApp flotante

